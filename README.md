word2vec stack으로 바꾸어 GAN을 돌립니다.
논문 : https://arxiv.org/pdf/1904.02293.pdf
현재 Mode Collapse 문제가 발생하고 있습니다. 해결방법을 모색중입니다.

Implements of GAN2vec using Keras
